# **App Name**: AssistLink

## Core Features:

- Real-time Customer Chat: Enables customers and support agents to exchange messages in real time within a dedicated chat widget.
- Agent Dashboard: A web interface for support agents to manage incoming chats, view conversation history, and send replies.
- Basic Chat Widget Customization: Allows users to choose the primary color for their embedded chat widget to align with brand aesthetics.
- Chat History Storage: Securely stores all chat logs and user data to maintain context across customer interactions.
- Enhanced Widget Embed: Provides comprehensive API endpoints for generating chat widget embed code in multiple programming languages (e.g., React, HTML), making integration flexible for developers.
- Agent Smart Reply Suggestions: Uses AI to suggest relevant quick replies to agents based on the customer's ongoing message content, serving as a productivity tool.
- Basic Chat Metrics: Provides simple analytics like the number of active chats and resolved conversations to give basic insights into support operations.
- Email Notifications: Configures and sends email notifications to agents for new incoming chat messages.
- AI-Powered Color Extraction: An optional tool that allows users to input their website URL, leveraging AI to automatically detect and suggest dominant brand colors for widget customization.
- Marketing & SEO Landing Page: A fully functional and SEO-optimized landing page to attract new users and provide information about the platform, including branding for the free tier ('Powered by AssistLink' message).

## Style Guidelines:

- The palette is based on themes of reliability and clear communication, featuring a light scheme. The primary color is a sophisticated blue (#3333CC), chosen to evoke trust and professionalism, ensuring good contrast with lighter text and elements.
- The background color is a subtle, light lavender-grey (#F0F0F4). It is visually aligned with the primary hue but is highly desaturated and bright to provide a clean and uncluttered canvas for chat interactions.
- An accent color of vibrant cyan-blue (#1FBAF5) is used for call-to-action buttons, interactive elements, and key highlights. It creates a striking contrast with both the primary and background colors to draw attention effectively.
- The main font for all text, including headlines and body, is 'Inter' (sans-serif). Its modern, objective, and highly readable design ensures clear communication for both agents and customers across all platforms.
- Use clear, universally recognizable icons that complement the clean, modern aesthetic. Prioritize icons that intuitively communicate chat actions (e.g., send, attach) and support states (e.g., online, busy).
- A responsive and adaptive layout ensures a seamless user experience. Chat widgets will be compact and efficient on smaller screens, expanding naturally on larger displays, maintaining an intuitive conversational flow.
- Subtle, smooth animations are used for message delivery and chat widget state changes (e.g., opening/closing). These brief transitions enhance user feedback without being distracting.
